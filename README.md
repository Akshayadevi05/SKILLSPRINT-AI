# SkillSprint AI

SkillSprint AI is an AI-powered career development platform designed to help users enhance their professional skills and prepare for future opportunities. The platform provides interview preparation resources, personalized learning roadmaps, career recommendations, and progress tracking through a modern and user-friendly interface.

## Features

* User Registration and Login
* Personalized Dashboard
* Interview Preparation Module
* Learning Roadmap Planner
* Career Recommendations
* User Profile Management
* Responsive and Modern UI

## Tech Stack

* React.js
* Vite
* JavaScript
* CSS

## Installation

```bash
npm install
npm run dev
```
